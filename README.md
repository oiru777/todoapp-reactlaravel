# Todoアプリ

## 使用技術
- バックエンド: Laravel 10.48.29（環境構築: Laravel Sail）
- フロントエンド: React 18.2.0
- UIライブラリ: Material UI
- 認証: Laravel Sanctum

## 機能
- タスク管理（追加、表示、編集、完了に切り替える、削除、ユーザーID紐付け）
- タグ絞り込み検索
- 認証（ユーザー新規登録、メール認証、ログイン、ログアウト）

## 追加予定機能
- パスワード変更

### 反省点
- ファイル構造->apiとwebで分けてない
- git管理->機能ごとにブランチを分けてない
